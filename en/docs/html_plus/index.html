HTML Content PLUS - Documentation

Created: 24/10/2018 Latest update: 10/04/2021 Version 3.0

The module allows you to display arbitrary HTML-code on site pages with the following options: «Expand / Collapse» long text; Control what screen resolutions the script will run on; Manage who will be given access to the content; Applying a custom template for a module.

Warning

The module was tested on a standard template and on a pure Opencart.
For each site, some improvements may be needed.

Installation

The first step can be performed automatically or manually.

1. In the admin panel go «Extensions > Extension Installer» and select the file «HTML_Content_Plus.ocmod.zip»;

OR

1. From the archive «HTML_Content_Plus.ocmod.zip» copy the folders «admin» and «catalog»;

2. In the admin panel go to«System > Users > User Groups» and set access rights for the administrator
(in the «Access Permission» and «Modify Permission» blocks, tick «extension/module/html_plus»);

3. In the list of modules «Extensions > Extensions» install the module «HTML Content Plus».

Module settings

To configure the module, go to: «Extensions > HTML Content - PLUS»

- Module name - name that will be displayed in the admin panel;

- Status - enabling / disabling the module;

- Block height for the visible description. The value is specified in pixels;

- Animation speed - the time it takes for a block with full text to unfold in milliseconds;

- Activate for screen resolutions - at what screen resolutions will the text «Expand / Collapse» script be activated;

- Heading level - you can choose the level from h1 to h6;

- Access - who will see this block: for all users / only for registered users / only for guests;

- Template - you can create your own module template based on the standard template «catalog\view\theme\default\template\extension\module\html_plus.twig» and place your own in the folder «catalog\view\theme\Your theme\template\custom_templates\module\html_plus\» after creating it, you can select it.

Appearance

If the standard button design «Expand / Collapse» does not fit into your site design, you can adjust the CSS styles in the «html_plus.css» file, which is located at:

« catalog \ view \ javascript \ jquery \ readmore \ html_plus.css »
