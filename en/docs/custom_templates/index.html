Custom Templates - Documentation

Created: 10/06/2021 Version 1.0

Allows you to set a specific template for a product, category, manufacturer (brand) or information pages. You can also assign one template for all products under a specific category, and also, if you wish, override this template for a specific product.

Warning

Please note, that when TEMPLATE is referred to, it's the single template files in your theme NOT the entire theme. You cannot set different themes per item with this extension.

If you installed a third-party theme on Opencart 3.x with the file extension «.tpl»
The modifier will NOT work.

The created files must have the file extension «.twig»

Installation

1. In the admin panel go «Extensions > Extension Installer» and select the file «Custom_Templates.ocmod.zip»;

2. In the admin panel go to«System > Users > User Groups» and set access rights for the administrator
(in the «Access Permission» and «Modify Permission» blocks, tick «extension/module/custom_templates»);

3. In the list of modules «Extensions > Extensions» install the module «Custom_Templates».

Using a modifier

After installation, the «custom_templates» folder should appear on your server along this path:
«catalog\view\theme\default\template\custom_templates\».

If you are using a different theme, you need to create this folder and additional folders in it «category», «information», «manufacturer», «product».

- Place the file with the changes made in a specific folder (the file name must be in Latin and with the extension «.twig»);

- In the Admin panel, go to the section for which you created the template (products, categories, white papers, manufacturers);

- Switch to editing mode and in the "Design" tab, select a template from the list;

- In a category, you can assign a template for all products that will be in this category;

- In products, you can assign a new or reassign a template;

- n manufacturers and in informational articles, you can assign a new template.
