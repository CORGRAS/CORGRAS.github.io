Introduction

This modifier will allow you to respond to customer reviews. Also, the modifier will allow you to:

  • To leave the response to a reviews;
  • Quickly change the status of the revocation;
  • The notification message of the client about the response by mail;
  • Added field of e-mail, to notify clients;
  • To change date of adding of a response (for versions is lower 2.3.x);
  • Enable / disable moderating reviews;
  • Enable / disable mandatory e-mail filling;
  • WYSIWYG editor for editing reviews.

Installation

Installation of the modifier consists of two parts. In each folder «Opencart v2.x» you will find two folders: «Step-1-Administration» и «Step-2-Templates».

  1. In the folder «Step-1-Administration» is the archive «extra_description_category_backend.ocmod.zip» which contains the main functional, localization and scripts.
  2. In the folder «Step-2-Templates» Find a list of modifiers for your template. The list will gradually be replenished for popular templates.

Step 1

  1. In the admin panel, go to «Extensions > Extension Installer» and install the file «extra_description_category_backend.ocmod.zip»;
  2. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;
  3. If the functionality does not appear, check the error log.

In the archive is a SQL file. If the database prefix is different from the standard one «oc_», then before the installation, extract the file «install.sql» and replace the prefix with a text editor «oc_» on your prefix, then upload the file back to the archive.


Step 2

Select the desired modifier from the folder «Step-2-Templates» and follow these steps:

  1. In the admin panel, go to «Extensions > Extension Installer» and install the file «extra_description_category_frontend.ocmod.xml» for your template;
  2. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;
  3. If the functionality does not appear, check the error log.

Settings

For each category description, there are your settings to go to: «Catalog > Categories > Your category» inside go to the «Design» tab, there you will find the settings:

  • Read more «Expand / Collapse» - Enable / disable separator;
  • The height of the visible part of the text - The height of the block of the visible description. The value is specified in pixels;
  • Hide description with 2+ pages - Shows the description only on the first page of the category;
  • Button «Read more» (Mobile) - For phones and tablets, an additional description is hidden completely and only the «Show Description» button is shown.

Design

If the standard design does not fit into your website design, you can modify the CSS styles in the «readmore.css», file that is located:

catalog \ view \ javascript \ jquery \ readmore \ readmore.css

Language

The modifier has full support for several languages. English and Russian are installed by default.
Below you will find instructions on how to add and translate a modifier into your language.

Depending on the version of OpenCart, the folder may be called «english» or «en-gb»


Admin-panel:

  • Copy the file «extra_category_description.php» from the folder:

    admin \ language \ russian \ catalog \ extra_category_description.php

  • Paste into the folder and then transfer the contents of the file:

    admin \ language \ YOUR_LANGUAGE \ catalog \ extra_category_description.php


Store Front:

  • Copy the file «extra_category_description.php» from the folder:

    catalog \ language \ russian \ product \ extra_category_description.php

  • Paste into the folder and then transfer the contents of the file:

    catalog \ language \ YOUR_LANGUAGE \ product \ extra_category_description.php
