Introduction

This modifier will allow you to respond to customer reviews. Also, the modifier will allow you to:

  • To leave the response to a reviews;
  • Quickly change the status of the revocation;
  • The notification message of the client about the response by mail;
  • To change date of adding of a response (for versions is lower 2.3.x);
  • Enable / disable moderating reviews;
  • Enable / disable mandatory e-mail filling;
  • WYSIWYG editor for editing reviews.

Installation

Installation of the modifier consists of two parts. In each archive«Install Opencart» you'll find modifiers:

  1. «ReplyToReviews_AdminPanel.ocmod.zip» - contains the main functionality for the Admin Panel.

    In the archive is a SQL file. If the database prefix is different from the standard one «oc_», then before the installation, extract the file «install.sql» and replace the prefix with a text editor «oc_» on your prefix, then upload the file back to the archive.

  2. «ReplyToReviews_StoreFront_Default.ocmod.zip» - Functional for the default template.
  3. «ReplyToReviews_StoreFront_Journal.ocmod.zip» - Functionality for the Journal Template.

Step 1

  1. In the admin panel, go to «Extensions > Extension Installer» and install the file «ReplyToReviews_AdminPanel.ocmod.zip»;
  2. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;
  3. If the functionality does not appear, check the error log.

Step 2

Choose a modifier for your template and follow these steps:

  1. In the admin panel, go to «Extensions > Extension Installer» and install the file «ReplyToReviews_StoreFront_*.ocmod.zip» for your template;
  2. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;
  3. If the functionality does not appear, check the error log.

Upgrade from v2 to v3

To update the version of the modifier, follow these steps.:


Step 1

  1. In the admin panel, go to «Extensions > Modifications» and delete the modifier Answer to the commentary;
  2. Unzip the archive«Update Opencart»;
  3. In the admin panel, go to «Extensions > Extension Installer» and install the file «ReplyToReviews_AdminPanel.ocmod.zip»;
  4. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;
  5. If the functionality does not appear, check the error log.

In the archive «ReplyToReviews_AdminPanel.ocmod.zip» is a SQL file. If the database prefix is different from the standard one «oc_», then before the installation, extract the file «install.sql» and replace the prefix with a text editor «oc_» on your prefix, then upload the file back to the archive.

  2. After the message of the successful installation in the «Extensions > Modifications» and click on the blue «Refresh» button (top right corner) See example;

Settings

Для настройки перейдите: «Setting > Store Settings» inside go to the «Options» tab, in the«Reviews» section you will find additional settings:

  • Disable moderation - Disable approval of reviews by the administrator;
  • Required field Email? - Required field for email;

Design

If the standard design does not fit into your website design, you can modify the CSS styles in the «readmore.css», file that is located:

catalog \ view \ javascript \ jquery \ readmore \ readmore.css

Language

The modifier has full support for several languages. English, Russian and Ukrainian are installed by default.
Below you will find instructions on how to add and translate a modifier into your language.

Depending on the version of OpenCart, the folder may be called «english» or «en-gb»


Admin-panel:

  • Copy the file «reply_reviews.php» from the folder:

    admin \ language \ english \ catalog \ reply_reviews.php

  • Paste into the folder and then transfer the contents of the file:

    admin \ language \ YOUR_LANGUAGE \ catalog \ reply_reviews.php


Mail:

  • Copy the file «reply_reviews.php» from the folder:

    admin \ language \ english \ mail \ reply_reviews.php

  • Paste into the folder and then transfer the contents of the file:

    admin \ language \ YOUR_LANGUAGE \ mail \ reply_reviews.php


Store Front:

  • Copy the file «reply_reviews.php» from the folder:

    catalog \ language \ english \ product \ reply_reviews.php

  • Paste into the folder and then transfer the contents of the file:

    catalog \ language \ YOUR_LANGUAGE \ product \ reply_reviews.php
