Текстовый блок HTML PLUS - Документация

Создан: 24/10/2018 Последнее обновление: 10/04/2021 Версия 3.0

Модуль позволяет отображать произвольный HTML-код на страницах сайта с возможностями: «Развернуть / Свернуть» длинный текст; Управлять на каких разрешений экранов будет работать скрипт; Управлять кому будет предоставлен доступ к контенту; Применение кастомного шаблона для модуля.

Предупреждение

Модуль тестировался на стандартном шаблоне и на чистом Opencart.
Для каждого сайта могут понадобиться некоторые доработки.

Установка

Первый шаг можете выполнить автоматически или вручную.

1. В админке перейдите «Модули > Установка расширений» и выберите файл «HTML_Content_Plus.ocmod.zip»;

ИЛИ

1. Из архива «HTML_Content_Plus.ocmod.zip» скопируйте папки «admin» и «catalog»;

2. В админ-панели перейти в «Система > Группы пользователей» и для администратора установить права доступа
(в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/html_plus»);

3. В списке модулей «Расширения -> Модули» установите модуль «Текстовый блок - HTML Plus».

Настройка модуля

Для настроики модуля перейдите: «Модули > Текстовый блок HTML - PLUS»

- Название модуля - название, которое будет отображаться в админ-панели;

- Статус - включение/выключение модуля;

- Высота блока для видимого описания. Значение указывается в пикселях;

- Скорость анимации - время, за сколько миллисекунд будет разворачиваться блок с полным текстом;

- Активировать для разрешений - при каких разрешений экрана будет активирован скрипт «Разворачивания / Свертывания» текста;

- Уровень заголовка - можете выбрать уровень от h1 до h6;

- Доступ - кому будет виден данный блок: для всех пользователей / только для зарегистрированных / только для гостей;

- Шаблон - можете создать свой шаблон модуля, взяв за основу стандартный шаблон «catalog\view\theme\default\template\extension\module\html_plus.twig» и разместить свой в папку «catalog\view\theme\Ваша тема\template\custom_templates\module\html_plus\» после создания можете выбрать его.

Внешний вид

Если стандартный дизайн кнопки «Развернуть / Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «html_plus.css», который находится:

« catalog \ view \ javascript \ jquery \ readmore \ html_plus.css »

Если у вас возникли проблемы, то можете ознакомиться с их решениями в этой статье «Решение проблем при установке модулей». Если не нашли ответы на свои вопросы можете связаться с нами.
