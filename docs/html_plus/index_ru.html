Введение

Модуль позволяет выводить на страницах сайта произвольный HTML-код.

  • Имеет визуальный редактор HTML;
  • Мультиязычный;
  • Через модуль можно управлять: скоростью «Развернуть / Свернуть» описания, высотой видимой части текста;
  • Для описания есть возможность скрыть длинное описание с появление кнопки «Развернуть / Свернуть».

Совместимость версий OpenCart

Opencart
3.x

Установка

  1. Из папки «upload» скопируйте папки «admin» и «catalog»;
  2. В админ-панели перейти в «Система > Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой extension/module/html_plus);
  3. В списке модулей «Расширения -> Модули» установите модуль Текстовый блок - HTML Plus;
  4. В «Дизайн -> Схема» выберите страницу магазина и расположите модуль в нужной позиции.

Настройка модуля

  • Название модуля - название которое будет отображаться в админ-панели
  • Статус - включение/выключение модуля
  • Разделитель текста «Развернуть / Свернуть» - Включение / выключение разделителя;
  • Высота блока - Высота блока видимого описания. Значение указывается в пикселях;
  • Скорость анимации - Время за сколько миллисекунд будет разворачиваться блок с полным текстом.

Внешний вид

Если стандартный дизайн не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «html_plus.css», который находится:

catalog \ view \ javascript \ jquery \ readmore \ html_plus.css

Локализация

Модификатор имеет полную поддержку нескольких языков. По умолчанию устанавливается английский и русский язык.
Ниже вы найдете инструкцию, как добавить и перевести модификатор на свой язык.

В зависимости от версии OpenCart папка может называться «russian» или «ru-ru»


Админ-панель:

  • Скопируйте файл «html_plus.php» из папки:

    admin \ language \ russian \ module \ html_plus.php

  • Вставьте в папку и после чего переведите содержимое файла:

    admin \ language \ ВАШ_ЯЗЫК \ module \ html_plus.php
