Introduction

The module allows you to display arbitrary HTML code on the site’s pages.

  • Has an HTML editor;
  • Multilingual;
  • Through the module you can control: the speed "Expand / Collapse" of the description, the height of the visible part of the text;
  • For a description, you can hide a long description with the appearance of the "Expand / Collapse" button.

Compatibility of versions of OpenCart

Opencart
3.x

Installation

  1. From the «upload» folder, copy the «admin» and «catalog» folders
  2. In the admin panel, go to «System > User Groups» and set the access rights for the administrator (in the «Allow browsing» and «Allow to edit» tick the extension/module/html_rm);
  3. In the module list «Extensions -> Modules» install the HTML Content Plus module;
  4. In «Design -> Layout» select the store page and position the module in the desired position.

Settings

  • Module Name - name that will be displayed in the admin panel.
  • Status - turn on / off the module.
  • Text Expand / Collapse Separator - Turn on / off the separator;
  • Block height - The height of the block of the visible description. The value is in pixels;
  • Animation speed - The time in how many milliseconds a full text block will unfold.

Design

If the standard design does not fit into your site design, you can correct the CSS styles in the file «html_plus.css», which is:

catalog \ view \ javascript \ jquery \ readmore \ html_plus.css

Language

The modifier has full support for several languages. English and Russian are installed by default.
Below you will find instructions on how to add and translate a modifier into your language.

Depending on the version of OpenCart, the folder may be called «english» or «en-gb»


Admin-panel:

  • Copy the file «html_rm.php» from the folder:

    admin \ language \ english \ module \ html_plus.php

  • Paste into the folder and then transfer the contents of the file:

    admin \ language \ YOUR_LANGUAGE \ module \ html_plus.php
