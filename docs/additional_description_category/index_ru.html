Введение

Данный модификатор поможет вам добавить текст, баннер или HTML-код в нижней части страниц категорий, который будет отличаться от описания категорий по-умолчанию.

  • Модификатор добавляет дополнительное текстовое поле в Админ-панели при добавлении новой категории или редактировании существующей;
  • Имеет визуальный редактор HTML, так что вы можете использовать HTML теги для SEO;
  • Дополнительное описание будет отображаться в нижней части списка продуктов;
  • Мультиязычный;
  • Через модуль можно управлять: названиями кнопок, иконками в кнопке, скоростью «Показа / Скрытия» описания и установкой / удалением необходимых дополнительных полей в Базу Данных;
  • Имеет функцию скрывать описание и дополнительное описание со 2+ страниц категорий;
  • Для стандартного описания и для дополнительного описания есть возможность скрыть длинное описание с появление кнопки «Развернуть / Свернуть»;
  • Настройки стандартного описания и дополнительного описания для каждого пункта категории.

Установка

Установка модификатора состоит из двух частей.

  1. «AD_Category_AdminPanel.ocmod.zip» - содержит основной функционал для Админ-панели, локализация и необходимые скрипты.
  2. «AD_Category_FrontEnd_Default.ocmod.zip» - модификатор для шаблона по-умолчанию.

Список постепенно будет пополняться для популярных шаблонов

Шаг 1

  1. В админке перейдите «Модули > Установка расширений» и установите файл «AD_Category_AdminPanel.ocmod.zip»;
  2. Получаем сообщение об успешной установке и идем в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу См. пример;
  3. В списке модулей «Модули -> Модули» установите модуль «Дополнительное Описание Категории»

    ВАЖНО!!!
    При первой установке модуля, автоматически устанавливаются дополнительные поля в таблицы БД. На всякий случай делайте бэкап БД

  4. Настройте модуль (Если статус модуля отключен - будут работать настройки по-умолчанию);
  5. Перейдите «Каталог -> Категории» и в нужной категории вставьте текст и настройте отображение

Шаг 2

Выберите нужный модификатор для вашего шаблона:

  1. В админке перейдите «Модули > Установка расширений» и установите файл «AD_Category_FrontEnd_Default.ocmod.zip»;
  2. Получаем сообщение об успешной установке и идем в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу См. пример;
  3. Возникли проблемы / ошибки, проверяем лог на ошибки.

Настройки

Настройки модуля

Для описаний категорий, есть настройки по-умолчанию, чтоб их настроить перейдите: «Модули > Дополнительное Описание Категории».
Во вкладке «Общие»:

  1. Можете назначить текст для состояния когда блок «Развернут» и когда блок «Свернут»
  2. Можете назначить иконки
  3. Скорость анимации при развертывании и свертывании блока
  4. Статус отвечает за включение настроек

Во вкладке «Статус БД» - можете проверить установлены нужные поля в таблице или нет. Также можете их удалить или установить.


Настройка категории

Для каждого описания категорий, есть свои настройки, чтоб их настроить перейдите: «Каталог > Категории > Ваша созданная категория» внутри перейдите во вкладку «Дизайн», там найдете настройки:

  • Разделитель текста «Развернуть / Свернуть» - Включение / выключение разделителя;
  • Высота видимой части текста - Высота блока видимого описания. Значение указывается в пикселях;
  • Скрывать описание со 2+ страниц - Показывает описание только на первой странице категории;
  • Кнопка «Показать описание» - Для телефонов и планшетов допольнительное описание скрывается полностью и показывается только кнопка «Показать описание»

Внешний вид

Если стандартный дизайн не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «readmore.css», который находится:

catalog \ view \ javascript \ jquery \ readmore \ readmore.css

Локализация

Модификатор имеет полную поддержку нескольких языков. По умолчанию устанавливается английский и русский язык.
Ниже вы найдете инструкцию, как добавить и перевести модификатор на свой язык.


Админ-панель:

  • Скопируйте файл «ad_category.php» из папки:

    admin \ language \ ru-ru \ catalog \ ad_category.php

  • Вставьте в папку и после чего переведите содержимое файла:

    admin \ language \ ВАШ_ЯЗЫК \ catalog \ ad_category.php


Магазин:

  • Скопируйте файл «extra category description.php» из папки:

    catalog \ language \ ru-ru \ product \ ad_category.php

  • Вставьте в папку и после чего переведите содержимое файла:

    catalog \ language \ ВАШ_ЯЗЫК \ product \ ad_category.php
