Дополнительное описание категориям - Документация

Создан: 13/11/2017 Последнее обновление: 27/12/2018 Версия 1.1

Данный модификатор поможет вам добавить текст, баннер или HTML-код в нижней части страниц категорий, который будет отличаться от описания категорий по-умолчанию. Также модификатор имеет возможность скрыть часть длинного текста для верхнего и нижнего описания.

Предупреждение

Модификатор тестировался на стандартном шаблоне и на чистом Opencart.
Для каждого сайта могут понадобиться некоторые доработки.

Установка

Установка модификатора состоит из двух частей.

«AD_Category_AdminPanel.ocmod.zip» - содержит основной функционал для Админ-панели, локализация и необходимые скрипты.

«AD_Category_FrontEnd_Default.ocmod.zip» - модификатор для шаблона сайта по-умолчанию.

ВАЖНО!!!

При первой установке модуля, автоматически устанавливаются дополнительные поля в таблицы БД. На всякий случай делайте бэкап БД

1. В админке перейдите «Модули > Установка расширений» и установите файл «AD_Category_AdminPanel.ocmod.zip»;

2. Получаем сообщение об успешной установке и идем в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу [скриншот];

3. В админке перейдите «Модули > Установка расширений» и установите файл «AD_Category_FrontEnd_Default.ocmod.zip»;

4. Получаем сообщение об успешной установке и идем в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу [скриншот];

5. Перейдите в список модулей «Модули > Модули» и установите модуль «Дополнительное Описание Категориям»

Настройка модуля

Для описаний категорий, есть настройки по-умолчанию, чтоб их настроить перейдите:
«Модули > Дополнительное Описание Категориям»

1. Во вкладке «Общие»:

- Можете назначить текст для состояния когда, блок «Развернут» и когда блок «Свернут»;

- Можете назначить иконки;

- Скорость анимации при развертывании и свертывании блока;

- Статус отвечает за включение настроек;

2. Во вкладке «Статус БД» - можете проверить установлены нужные поля в таблице или нет. Также можете их удалить или установить.

Настройка в категориях

Для каждого описания категорий, есть свои настройки, чтоб их настроить:

Внутри категории перейдите во вкладку «Дизайн», там найдете настройки.

- Разделитель текста «Развернуть / Свернуть» - Включение / выключение разделителя;

- Высота видимой части текста - Высота блока видимого описания. Значение указывается в пикселях;

- Скрывать описание со 2+ страниц - Показывает описание только на первой странице категории;

Внешний вид

Если стандартный дизайн кнопки «Развернуть / Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «readmore.css», который находится:

« catalog \ view \ javascript \ jquery \ readmore \ readmore.css »
